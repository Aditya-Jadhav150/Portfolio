Aegis AI

Deep Learning / Synthetic Media Detection

The Problem

The exponential growth of sophisticated generative AI and diffusion pipelines (like Midjourney and Latent Diffusion) has made synthetic media virtually indistinguishable from authentic photos, presenting severe security, identity, and misinformation risks.

The Solution

An enterprise-grade forensic platform featuring a decoupled web service architecture. It combines advanced computer vision models (convolutional nets, auto-encoders, vision transformers) with classical digital image forensics to isolate synthetic face structures.

Key Features

  • Production-Grade XGBoost Fusion Engine extracting 9 distinct biometric and digital artifacts (e.g., Frequency Anomaly, Noise Residuals).
  • Deep Multi-Modal Cross-Attention Network utilizing Swin-T, ResNet-18, and Latent CNN branches optimized via Compound Contrastive Loss.
  • Automated MTCNN face alignment pipelines and strict security protocols including Google OAuth SSO and Hugging Face proxy management.
Technologies Used
Python PyTorch Flask XGBoost Hugging Face
Outcome & Impact

Delivered highly precise classification on GAN/Diffusion fakes using multi-branch analysis, successfully deployed via production Hugging Face Spaces.

EXONYX

Astronomy / AI Data Pipeline

The Problem

Processing raw photometric light curves from NASA's Kepler and TESS missions manually is labor-intensive, while separating true exoplanet transits from eclipsing binaries requires complex signal processing.

The Solution

An AI-powered autonomous platform built with Next.js and FastAPI that processes raw light curves, detects transit signals using TLS, and validates exoplanet candidates using a convolutional neural network (AstroNet).

Key Features

  • Autonomous processing pipeline featuring WOTAN detrending and Transit Least Squares (TLS).
  • Deep learning validation running AstroNet CNN on phase-folded light curves.
  • Interactive 3D scientific transit simulator using Three.js and React-Three-Fiber.
Technologies Used
Next.js FastAPI Python Three.js
